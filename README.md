# hackaton_test

run the virtual environment
```bash
pipenv shell
```

Set env vbles:
```bash
export FLASK_ENV=development
export FLASK_APP='.'
```

run:
```bash
flask run --host=0.0.0.0 --port=3000
```
or
```
python3 app.py
```

# requirements
```bash
pip install flask
pip install google-cloud-firestore
pip install pandas
pip install plotly
pip install entsoe-py
```
